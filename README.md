## Run Locally

Clone the project

```bash
  git clone git@github.com:Tonips22/utools-react.git
```

Go to the project directory

```bash
  cd utools-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
